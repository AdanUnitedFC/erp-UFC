# UFC Intrasys

ERP para forwarding. Fullstack en FastAPI + React + PostgreSQL.

## Backend

- FastAPI + JWT Auth
- `/token`: login

## Frontend

- React + Vite (proximamente login UI)

## Scripts

- Levantar backend:
    cd backend && uvicorn app.main:app --reload

- Levantar frontend:
    cd frontend && npm install && npm run dev
